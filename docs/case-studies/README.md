# Production Case Studies

这个目录收集 auto-cli 在生产环境中的真实使用案例。

## 📋 已收集案例

目前收集中，敬请期待首批案例！

## 🎯 案例征集目标

我们希望收集 **5+ 生产环境使用案例**，以满足 CNCF Incubating 级别要求的"生产使用证明"。

## ✍️ 如何提交案例

### 方式 1: GitHub Discussions（推荐）

1. 访问 [GitHub Discussions](https://github.com/ktyyer/auto-cli/discussions)
2. 在 "Production Cases" 分类下创建新帖子
3. 按照 `TEMPLATE.md` 填写案例内容
4. 我们会整理后添加到本目录

### 方式 2: Pull Request

1. 复制 `TEMPLATE.md` 为 `your-company-name.md`
2. 填写完整案例信息
3. 提交 PR 到 `docs/case-studies/` 目录
4. 通过审核后合并

## 📝 案例要求

### 必填信息

- 公司/项目名称
- 项目类型（Web 应用 / CLI 工具 / 库 / 其他）
- 主要使用场景
- 实际效果（定量或定性）

### 可选信息

- 技术栈详情
- 具体使用示例
- 团队规模
- 时间/成本节省数据
- 质量改进指标

## 🎁 早期采纳者权益

前 10 个提交案例的用户将获得：

1. **README 展示** - 在项目 README 中展示你的公司/项目名称和链接
2. **Early Adopter 徽章** - 在案例页面显示特殊徽章
3. **优先功能建议权** - 你的功能需求将优先考虑
4. **官方推广** - 案例将在官方渠道（Twitter/LinkedIn/博客）推广
